# Exchange rate calculator monorepo

## Installation steps
```shell
# This step is optional
cp .env.example .env

# Install all the dependencies
yarn

# To start whole monorepo in the development mode (hot reload support)
yarn dev
```

## Global tech stack
- typescript
- eslint
